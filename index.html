Drone simulator with 3D Gaussian Splats

By Antoine Keller. Code on Github.

Keyboard controls
  • WASD - Move
  • Arrow Keys - Rotate
  • F - Toggle Free Mode
  • R - Reset Camera
Gamepad controls
  • Left stick Y - Up/Down
  • Left stick X - Yaw angle
  • Right stick - Navigate
  • R2 - Toggle Camera Mode
  • L1/L2 - Camera pitch
